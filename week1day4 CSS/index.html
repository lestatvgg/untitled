HTML

        
 HTML (от англ. HyperText Markup Language — «язык гипертекстовой разметки») — стандартизированный язык разметки документов для просмотра веб-страниц в браузере.
            Веб-браузеры получают  документ от сервера по протоколам HTTP/HTTPS или открывают с локального диска,
            далее интерпретируют код в интерфейс, который будет отображаться на экране монитора. 

            В HTML можно встроить программый код на языке программирования JavaScript, для управления поведением и содержанием веб-страниц.
            Также включение CSS в HTML описывает внешний вид и макет страницы.
